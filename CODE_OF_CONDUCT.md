# Code of Conduct

## Our commitment

We want Handle to be a welcoming, respectful, and constructive community for everyone.

We expect all participants in this project—including maintainers, contributors, and community members—to treat each other professionally and respectfully in issues, pull requests, discussions, and any other project spaces.

## Expected behavior

Examples of expected behavior include:
- being respectful and considerate in communication
- giving constructive, actionable feedback
- assuming good intent while still communicating clearly
- being open to disagreement without becoming hostile
- focusing discussion on what is best for the project
- respecting different levels of experience

## Unacceptable behavior

Examples of unacceptable behavior include:
- harassment, intimidation, or abuse
- personal attacks, insults, or inflammatory language
- discriminatory or derogatory comments
- trolling or deliberately disruptive behavior
- doxxing or sharing someone’s private information without permission
- repeated bad-faith arguments or refusal to engage constructively
- sexualized language or unwelcome sexual attention

## Scope

This Code of Conduct applies in all project spaces, including:
- issues
- pull requests
- discussions
- code review
- project-related chat or communication channels
- public spaces where someone is representing the project

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it to:

support@tonkotsu.ai

Please include as much relevant context as you can. We will review reports in good faith and handle them as promptly and discreetly as possible.

## Enforcement

Maintainers may take any action they deem appropriate in response to behavior that violates this Code of Conduct, including:
- warning the offender
- removing comments or content
- closing or locking threads
- rejecting contributions
- temporarily or permanently banning participants from project spaces

## Maintainer responsibility

Maintainers are responsible for clarifying and enforcing this Code of Conduct and may moderate content or participation when necessary to protect the health of the project community.

## Changes

Maintainers may update this Code of Conduct over time as the project evolves.
